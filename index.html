Welcome to Complex_Bessel.

Complex_Bessel is a C++ interface/library that allows the evaluation of the Bessel functions. The computations are performed by FORTRAN code developed by D.E. Amos. The original source code is available in the SLATEC library.

Authors and Contributors

The library is developed by Joey Dumont (@joeydumont) and Denis Gagnon (@McNoggins).
